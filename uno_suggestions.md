# UNO Blazor — Feature Suggestions List

---

## 🎮 Gameplay Features
1. **Tournament mode** — bracket-style multi-room competition with automatic bracket progression
2. **Spectator mode** — join a room as a watcher with no hand, just observing live
3. **Rematch button** — quick-rematch from the game-over screen without returning to lobby
4. **Private messaging** — whisper to specific players in the lobby chat
5. **Jump-In visual timer** — visible countdown bar on-screen during the jump-in window
6. **Card trading rule** — optional rule: players can trade one card per round
7. **Team mode** — 2v2 or 3v3 with shared win condition
8. **Revenge card** — special optional rule: loser of last round draws 0 to start
9. **AFK auto-kick timer** — kick idle players after X seconds with a visible countdown

---

## 🤖 CPU / AI Improvements
10. **CPU difficulty names visible in lobby** — show Easy / Medium / Hard / Savage labels per CPU slot
11. **CPU "thinking" delay animation** — animated "..." bubble while CPU processes its turn
12. **CPU taunts** — CPU sends contextual chat messages ("UNO already? 😏")
13. **Adaptive CPU** — CPU adjusts difficulty mid-game based on how far behind they are

---

## 👤 Profile & Social
14. **Friend system** — add friends, see who's online, invite directly to a room
15. **Player titles / badges** — unlock cosmetic titles ("Card Shark", "UNO Master") based on stats
16. **Achievement system** — e.g. "Win with 1 card left 5 times", "Stack Draw-4 three times"
17. **Avatar frames** — decorative borders around avatars earned via achievements
18. **Profile visit** — click a player's name in lobby or leaderboard to view their public profile
19. **Match replay** — step-by-step card-by-card breakdown of past matches on the profile page

---

## 📊 Stats & Leaderboard
20. **Per-rule stats** — track wins/losses separately for Classic vs custom rule sets
21. **Leaderboard filters** — filter by time period (This Week / This Month / All Time)
22. **CPU leaderboard toggle** — show/hide AI bots from the global leaderboard
23. **Head-to-head stats** — see your personal record against a specific player
24. **Most-played card stat** — which card you play most often (Draw 2, Skip, etc.)

---

## 🎨 UI / UX
25. **Dark/light theme toggle** — alternative light mode for the UI
26. **Card back themes** — unlockable card back designs (e.g. galaxy, fire, minimal)
27. **Table theme skins** — different game board backgrounds
28. **Sound effects volume slider** — separate music vs SFX sliders in a settings panel
29. **Confetti / celebration animation** — richer win screen effect
30. **Mobile-responsive layout** — optimized touch layout for phones/tablets
31. **Accessibility mode** — high-contrast cards, larger text, color-blind friendly card symbols

---

## 🔒 Multiplayer / Lobby
32. **Password-protected rooms** — optional room password for private games
33. **Room browser** — list of public open rooms to join without needing a code
34. **Lobby chat history** — persist chat scroll so late joiners can read previous messages
35. **Ready-check system** — host can't start until all players click "Ready"
36. **Reconnect support** — if a player drops mid-game, hold their seat for 30 seconds
37. **Player limit enforcement on join** — show "Room Full" clearly rather than silently failing
38. **Host migration** — if host disconnects mid-game, promote next player automatically

---

## 🛠️ Admin / Backend
39. **Match log viewer** — admin can browse all recent matches with full player results
40. **Broadcast announcement** — admin sends a server-wide notification (shown as a banner in-game)
41. **Reset player stats** — admin button to zero out a player's stats
42. **IP-based ban** — stronger ban that blocks new guest accounts from the same session

---

## 🚀 Platform / Technical
43. **Push notifications** — "Your friend just started a game!" browser notification
44. **PWA / installable app** — make it installable on desktop/mobile as a standalone app
45. **Discord Rich Presence** — show "Playing UNO Blazor · Room ABC123" in Discord status
46. **Share room link** — one-click copy of a full URL that auto-fills the room code on open
47. **Replay export** — download a match replay as a JSON or shareable link
